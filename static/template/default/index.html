Poki SDK

Once you have enabled the PokiSDK plugin, all the required files are setup for your project to make web-mobile builds.

Note

You can checkout the following files for example usage.
DemoScript.ts
PokiPlatform.ts

Generating Build

As mentioned earlier, you should be making web-mobile builds for the PokiSDK to work as expected. Once you have the build dialog, scroll down to the poki-sdk section where you will be prompted to provide the site lock code. This will be provided by your contact from Poki.

Warning

When a release build is generated, the sitelock code is added to the build which prevents you from testing locally. Please makde a debug build or use the preview button to test your code.
